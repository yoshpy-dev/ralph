# Evidence

Evidence is anything stronger than assertion.

Good evidence examples:
- test output
- lint or typecheck output
- structured logs
- screenshots
- traces or metrics
- precise diff references
- explicit walkthrough notes

Weak evidence examples:
- "it should work"
- "I am confident"
- "the code looks fine"
- "probably covered"

When reporting completion, prefer:
1. what was verified
2. how it was verified
3. what remains unverified
